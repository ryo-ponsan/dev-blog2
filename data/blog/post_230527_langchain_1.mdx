# 論文の概要を把握するテンプレート

社内向けに便利ツール紹介という名目で作りました。

Google Colab 上で OpenAI の API key と読み込みたい論文 PDF を入力し、  
「すべてのセルを実行」で試せます。ご自由にご活用ください。  
ちょっとコードやプロンプトを編集すれば、様々な情報を取得し、データを変換できます。  
対応外部データ詳細については [LangChain Indexes のドキュメント 👈](https://python.langchain.com/en/latest/modules/indexes/document_loaders.html)

## フロー

- 大量のテキストデータを保有する PDF を 分割
- エンベディング
- ベクトル化したデータを[ChromaDB](https://www.trychroma.com/) に保存
- プロンプトで指示
- 指示に対応する ChromaDB からベクトルを抽出しプロンプトに従い変換し出力

[論文概要把握ノートブックはこちら 👈](https://github.com/ryo-ponsan/LangChain-tools/blob/main/langchain_research_summarize_example.ipynb)
